python koudai48_win.py

Linux

# 使用方法
口袋:
 一.使用1查询成员id和房间号
 一.设置setting.conf文件内的user和password以及首推偶像的name,group,id和房间号
 二.第一次使用请选择8登陆,token未过期可不用反复登陆。

# 功能
        1.查看成员ID                    7.根据ID查询用户信息
        2.查看首推成员发言      8.登陆
        3.查看首推留言版信息    9.获取直播列表
        4.查看首推房间信息      10.每日签到
        5.查看个人房间列表      11.关注/取关小偶像
        6.获取好友ID                    12.关注/取关聚聚
